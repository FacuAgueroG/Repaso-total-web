Etiquetas semanticas estructurales

Div - Divisor sin semantica
header - cabecera de un sitio web o de una seccion
main- define la seccion princial del documento, solo puede existir una etiqueta main por documento
footer- pie de pagina de un sitio web o de una seccion
nav- representa una navegacion
Section - representa una seccion, puede contener al article
article - representa una seccion autocontenido (que por si sola se entiende)
Aside - Representa contenido complementario o secundario
adress - contacto con una pagina o un cliente

Etiquetas de bloque vs etiquetas de linea

div - Es una etiqueta de bloque, crea saltos con sus etiquetas hermanas y ocupa todo el ancho de la pantalla.
Span- es una etiqueta de linea que solo ocupa el espacio que esta utilizando
