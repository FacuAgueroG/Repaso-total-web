Se enseña como funciona el flex direciton que es la configuracion que le vamos a dar a nuestro texto para ponerlo en ciertos lugares de la pantalla y como lo vamos a mostrar sobre ella. Clase 05 Alineacion del cross axis

Esto explicac como funcionan algunas de las funcionalidades basicas como align-items y align-content. El primero sirve para trabajar especificamente sobre cada uno de los contenidos y el segundo sirve para trabajar sobre el total del contenido de una caja contenedora

Clase 06 flex-grow

El flex grow sirve para que los items (ITEMS, NO CONTENEDORES) ocupen el espacio vacio de un contenedor. Aun asi, a los items al estar por separados se les puede dar la opcion de ocupar mas o menos espacio dentro de estos

______________________________

Clase 07 flex-shrink

Cuando el navegador se achica y llega a ser lo suficientemente chico como para que se las cosas con flex tengan que achicarse esto le da o no permiso para hacerlo, PERO, ademas tambien puede ajustar item x item cuando tendran que achicarse con respecto a los otros

_____________________________

Clase 08 flex-basis y flex

Esto sirve para darle un tamaño a los items (todos por igual). Esto se dara ancho o alto dependiendo a donde apunte el flex-direction. 

flex es un shorthand de flex-grow flex-shrink y flex-basis

____________________________

Clase 09 Alineacion de hijos flexbox (order y align-self)

order sirve para ordenar objetos en un contenedor... ahora bien, cada item debe tener su propia clase para que esto suceda

self esto sirve para que podamos darle a un solo item alguna propiedad de align-items
